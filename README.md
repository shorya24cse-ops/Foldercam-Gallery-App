>> FolderCamGallery

A simple Android application that allows users to capture photos, browse images from a selected folder, view image details, and manage (delete) images easily.

>> Features
* Capture photos using the device camera
* Select any folder from the device
* View all images in a grid layout (gallery style)
* View image details:
  ~ Name
  ~ Path
  ~ Size
  ~ Date
* Delete images with confirmation dialog
